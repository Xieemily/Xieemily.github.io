LED屏幕滚动着下一班车的到站信息。

主角（画外）

红细胞通过各个动脉，穿行到下一个目的地。

人们进入地铁口。

主角（画外）

流过淋巴结的时候，免疫细胞会检查里面有没有不该存在的东西——

人们排队安检，安检员机器。

主角（画外）

病毒、

异物。

主角（画外）

主角（画外）

圆饼形，中间凹下去，像一个没有洞的甜甜圈。这是最高效的搬运形状，方便挤过毛细血管。

这个人的午饭上车了。

主角（画外）

到了毛细血管，偶尔，有一些蛋白质会漏出去。

主角（画外）

291. 站台另一侧 / 白天 / 手持镜头 — 采访女生

从背后追上一个年轻女生。

主角（画外）

请问，你有没有在地铁上丢过什么东西？

女生

丢东西……充电宝？一个粉色的充电宝，两万毫安的。

651. 站台另一侧 / 白天 / 手持镜头 — 采访另一个行人

行人摘下耳机。

行人

你说什么？

主角（画外）

你有没有在这站丢过什么东西？

行人

主角（画外）

我在调查这个地铁里消失的东西，收集一些信息。

467. 站台另一侧 / 白天 / 手持镜头 — 采访上班族，天堂伞凭空消失

一个中年女人，四十多岁，穿着通勤的外套，挎着一个深色的包。她非常认真。

上班族

那种深蓝色的，自动收的。我一直放包里，下雨天拿出来用，不下雨就在包里待着。用了四五年了。很好的伞，是那个牌子，叫什么来着，很有名——

她顿了一下，等对方补充，但没等到。

上班族

天堂伞。

上班族

我记得很清楚，因为那天天气预报说有雨。上了车我还看了一眼，就在侧边口袋。然后下车的时候——就没有了。

上班族

不是忘了。我不会忘。你知道吗？真的不见了。

主角（画外）

很多人都在这里丢了东西。

234. 地铁里 / 白天 / 手持镜头 — 保洁叫他抬脚

他坐在站台座椅上，手机架在膝盖上对着轨道方向在录。

保洁阿姨拖着拖把过来了。到他跟前停了一下。

3. 站台 / 白天 / 手持镜头 — 采访打工人，丢了一根香蕉

白领

香蕉。

主角（画外）

……香蕉？

白领

对。

停顿。

白领

我买回来放了两天一夜。那天早上，它变成了一根完美的香蕉。

主角（画外）

完美的香蕉。

白领

你要是看到它，马上就明白了。正好两天一夜，少会带绿色，多了就开始长黑点，但那天早上，就那天早上，我起来的时候，它就在那里，完美，没有一丝绿色，也没有一个黑点。我放在包里上的车。一定是被偷走了。没有生物能抗拒那么完美的香蕉。

主角（画外）

是哪天丢的？

白领

三天前。我下车之后发现没了，难受了一整天。

234. 地铁里 / 白天 / 手持镜头 — 保洁叫他抬脚

他坐在站台座椅上，手机对着轨道方向在录。

保洁阿姨拖着拖把过来了。到他跟前停了一下。

保洁

右脚抬一下。

他抬脚。她拖过去。

5. 站台座椅旁 / 白天 / 手持镜头 — 采访退休老师，袋子没破但鱼没了

一个退休老师。六十多岁，头发花白，穿着一件看不出颜色的薄外套。他坐在站台的塑料座椅上，腿上放着一个塑料袋。

语气很平，没有任何要把这件事讲成故事的意思。

退休老师

从菜场买的鲈鱼。活的。

他提了提手里的塑料袋，示意了一下。

退休老师

就这种袋子，扎好的，里面有水。上车的时候还能感觉到它在动。

他低头看了一眼袋子。

退休老师

到站下车，我就觉得轻了。一看——袋子还在，水还在。

他停了一下。不是为了制造悬念，是在确认自己要说的话。

退休老师

鱼没了。

安静了两秒。

退休老师

鲈鱼清蒸好吃。

他又看了一眼袋子，好像到现在还在确认这件事。

退休老师

就是没有了。

7. 主角的房间 / 夜 / 屏幕录制 — 回看监控，一把伞消失在两帧之间

电脑屏幕。一个文件夹，里面是密密麻麻的视频文件。文件名很长：

“0917-雨伞-中山公园.mp4” “0919-手套-虹桥路.mp4” “0921-不明-中山公园.mp4”

鼠标双击了一个文件。播放器打开。画面是地铁车厢内景，固定角度，像是手机架在窗边拍的。车厢里人不多。

画面左下角，一排座椅。其中一个座位上，放着一个东西。

鼠标拖动进度条。画面跳了一下。

那个东西——看不太清，但轮廓像一把折叠伞——立在座位上。

时间码：19:07:34。

鼠标继续拖。

时间码：19:07:36。

座位上什么都没有了。

他拖回去。19:07:34。拖过去。19:07:36。没了。

中间没有19:07:35。

他又拖回去。又拖过去。反复。鼠标在时间轴上来回，来回，来回。

他把画面暂停在19:07:34，放大。像素变粗了。那个东西的轮廓更清楚了一点——深蓝色，折叠的，自动收的那种。一把天堂伞，立在空荡荡的地铁座位上。

他截了图，拖进另一个文件夹。文件夹的名字叫"消失"。

浏览器打开了。搜索框。

他打字：“地铁 东西 消失”

搜索结果：一些失物招领的帖子、地铁投诉、一个关于"地铁能不能带金鱼"的知乎回答。

他往下翻。点开一个。退出来。又点开一个。

然后他换了一个关键词：“物体 消失 物理”

结果变了。第一页出现了一个科普帖子，标题很长，带着"黑洞"两个字。

他点进去。

页面加载出来。文字、配图、一张黑洞的模拟图。

他盯着这个页面。

几秒。

黑洞。

他目不转睛。黑洞似乎要把意识吸进去——

老妈（画外）

家里没有盐了！

老妈（画外）

柜子里也没了，你去楼下，那个小超市，买一包回来。

主角

我不去。

老妈

千万不要买错了，要那种的。

主角

哪种？

老妈

上次你买的那个不对。

主角

哪个？

老妈（画外）

就那个，你不要买错了。

9. 小超市 / 夜 / 手持镜头 — 买盐

货架。各种盐摆在一排。

他站着，旁边有人走过，货架间隙狭窄，他侧身让开。

让一个大爷。

让一个拿篮子的女人。

他排在长长的结账队伍后面，开始看手机。搜索：微型黑洞 出现概率

搜索结果里有一个视频缩略图：一只猴子坐在打字机前面。他点开了。

视频里，猴子在乱按键盘。字母在屏幕上随机跳出来。旁白说：“给猴子一台打字机和无限的时间，它能打出任何你想要的东西。”

他盯着猴子，意识到了什么。

他把页面往下滑——另一个词条：“意大利面条效应”，正要打开，

收银员

三块五。

他拿着盐从超市出来，走了几步，低头看一眼袋子。

又看了一眼。

他转身回去。

他又从超市出来，拿着另一包盐。

10. 站台 / 又一天 / 白天 / 手持镜头 — 保洁阿姨认出了他

他又坐在站台上。手机对着轨道方向在录。

保洁阿姨从远处推着拖把过来。走到他旁边的时候，她的步子慢了一下。

她看他一眼。他把两只脚都抬起来。

她拖过去了。

11. 菜场 / 白天 / 手持镜头 — 问鲈鱼价，买了便宜的黑鱼

喧闹。叫卖声。水产摊位前面，几个水池子里的鱼在翻。

主角

鲈鱼多少一斤？

鱼贩

三十八。

主角

那边的呢？

鱼贩

十五。

他指向那个池子。

12. 地铁车厢 / 白天 / 手持镜头 — 第一次带鱼坐地铁，鱼还在

他坐在座位上。镜头低着，对着他的腿和脚——他的膝盖上放着一个塑料袋，袋子里有水，有一条鱼。

列车到站，开门。关门。

一会后，地铁里人满为患，他手扶袋子，跟着人群摇晃。

一会后，一个胖子坐到他旁边，他被挤醒了。

一会后，傍晚，车厢空了，他昏昏欲睡。

门开，他下车，低头，看袋子。

13. 主角的家 / 客厅 / 夜 / 固定镜头 — 吃饭，老妈劝他去面试

饭桌上。一盘红烧鱼——做得不太好看，家常菜的样子。

老妈

你姨今天早上打电话来了，她同事那边在招人。

她夹了一块鱼，放碗里。

老妈

做什么的我忘了……好像是什么仓库管理的。我都给你约好了，周五早上去面一下啊，你天天在家待着，也没事干。

他的筷子停了一下。

老妈

这鱼刺多，下次别买了。

14. 菜场 / 又一天 / 白天 / 手持镜头 — 这次直接买鲈鱼，没问便宜的

同一个水产摊。他指另一个池子。

主角

要那个。

15. 地铁车厢 / 白天 / 手持镜头 — 第二次带鱼坐地铁，鱼还是没消失

同样的角度。膝盖上放着塑料袋。鱼在里面。

到站了。开门。关门。

又一站。开门。关门。

他低头。

又一站。他背后窗外，广告灯在显示视频。

又一站。

他拎着袋子下车了。

16. 站台 / 傍晚 / 手持镜头 — 袋子漏水，保洁大喊提醒

他坐在站台座位，抱着塑料袋。

他低头，缓缓打开，列车出站，发出轰鸣声。

保洁阿姨在远处，停下来，拄着拖把，显然注意到了什么，抬手指他的袋子，脸色惊慌。

保洁看着他，大声说了什么，声音被进站的车盖过去了，只看见她的面部动作。

声音减弱。她的声音浮现。

保洁

别往身上放！你那塑料袋儿漏水！！

他站起来。

裤子湿了一大片。

周围的行人都看着他。

17. 主角的家 / 厨房和客厅 / 夜 / 手持+固定镜头 — 摘豆角，说明天有面试

裤子挂在阳台上。

鱼在厨房水池子里。

厨房台面上，一盘豆子，一盘没剥皮的。他在掰豆角的丝。旁边是老妈，动作比他快。

老妈

明天的面试，千万不要忘了，人家介绍的，一定要好好面，一早就过去，不要晚了，早高峰人多，堵车，门口——。

主角

知道了妈。

老妈自己的剥完了，从他盘子里拿过来一把。

老妈

手慢。

18. 主角的房间 / 深夜 / 屏幕录制+手持 — 深夜反复看视频，决定出门

啪。关灯。房间暗了。

他躺在床上。

他翻了个身。

他打开手机。打开文件夹。点开那段视频。

那把伞。19:07:34。在。

19:07:36。没了。

他盯着屏幕。

关掉手机。

黑暗。

安静。

他又打开了手机。

这次他没有看视频。他打开浏览器——上次在超市没看完的那个页面还在。他点开了"意大利面条效应"。

一张示意图：一个人形被拉成一根长条。

他盯着那张图。

几秒。

关掉手机。

黑暗。

19. 地铁站 / 深夜 / 手持镜头 — 冲进地铁站赶最后一班车

镜头晃动。脚步声很快。他在跑。

地铁站的灯还亮着，但通道里几乎没有人了。他的呼吸声很重。画面乱成一团——天花板，地面，楼梯，墙上的指示牌，模糊的灯光。

他冲过闸机。

站台上空的。远处，隧道里传来列车的声音。灯光从隧道里漫出来。

车来了。

最后一班。

门开。

他走进去。

20. 车厢内 / 深夜 / 手持镜头 — 空车厢里看见那把伞，伸手触碰，画面黑屏

车厢是空的。只有他一个人。

白色的灯管。座椅一排一排，全是空的。

他坐下来。

列车开了。

他的呼吸慢慢平下来。

车窗外面黑的，什么都看不见。

然后——他注意到了什么。

对面的座位上。

一根香蕉。

镜头晃了一下——他的手在抖。

他站起来。

走过去。

伸出手。

手快要碰到那根香蕉了。

列车进了隧道。

车窗外面的黑暗突然被打破——广告灯箱一块一块闪过来。光从窗户涌进车厢，一闪一闪，红的，蓝的，白的，颜色很快，像在翻一本巨大的书。

灯光下，香蕉被举起来，翻面，翻回来，翻面，翻回来。

完美的黄色，从头到尾。

完美的香蕉。

光越来越密，画面开始抖。

信号出了问题——画面撕裂了一下，像视频文件损坏的那种撕裂。横条纹，色块，噪点。

然后黑了。彻底黑了。声音切断。

21. 主角的家 / 早晨 / 固定镜头 — 老妈发现房间没人

白天的光从窗帘缝隙里照进来。

走廊。老妈的拖鞋声。她走到他房间门口，推了一下门。

老妈

起来了。

房间里没有人。

床上被子掀开着，枕头上有个凹下去的印子。

老妈

走了啊？也不知道说一声。

22. 地铁站 / 早高峰 / 固定镜头 — 保洁捡起一根意大利面

早上八点的地铁站。

人群从各个入口涌进来。闸机滴滴响，一个接一个。脚步声叠在一起，变成一片连续的、低沉的轰响。LED屏幕上滚动着下一班车的到站信息。一切如常。

镜头是固定的。不再是他的手持视角。拿摄像机的人不在了。

保洁阿姨从画面右侧走进来。蓝色工服，拖把。她在擦站台的玻璃护栏。和之前每一次一样。

她擦到一个位置，停了一下，低头。

一根面条。湿的。在地铁站的地砖上。一根意大利面。

她弯腰捡起来，看了一眼。

她似乎发现了什么。

她把面条扔进垃圾桶。

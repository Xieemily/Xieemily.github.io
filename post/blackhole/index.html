黑洞

短片剧本 | 伪纪录片 | 约8-10分钟

 Posted on March 15, 2026  |   18 minutes  |   4618 words  |  Xieemily

短片剧本 | 伪纪录片 | 约8-10分钟

Script

1. 地铁站 / 白天 — 地铁是城市的血管

主角（旁白）

地铁是城市的血管。

LED屏幕滚动着下一班车的到站信息。

主角（旁白）

每天，几百亿个红细胞经过血管。

人们进入地铁口。

主角（旁白）

流过淋巴结的时候，免疫细胞会检查里面有没有不该存在的东西——

人们排队安检，安检员机器。

主角（旁白）

病毒、

主角（旁白）

细菌、

主角（旁白）

异物。

主角（旁白）

到了毛细血管，有一些蛋白质会漏出去。

掉在地上的甜甜圈，被咬了一口。

主角（旁白）

这些漏出去的蛋白质大部分会被淋巴系统回收。

一个工作人员用电脑打开地铁失物招领网站。

主角（旁白）

有一小部分，永远也回不来了。

保洁阿姨把垃圾倒进垃圾桶。

2. 站台另一侧 / 白天 / 手持镜头 — 丢背包

一个工人大哥。

主角（画外）

你有没有在这丢过什么东西？什么都行。

大哥

我丢过一个背包。就那种运动背包，可能是忘车上了吧。

3. 站台另一侧 / 白天 / 手持镜头 — 采访女生

从背后追上一个年轻女生。

主角（画外）

请问，你有没有在地铁上丢过什么东西？

女生

丢东西……充电宝？一个粉色的充电宝，两万毫安的。

4. 站台另一侧 / 白天 / 手持镜头 — 采访上班族，天堂伞凭空消失

一个中年女人，四十多岁，穿着通勤的外套，挎着一个深色的包。她非常认真。

上班族

那种深蓝色的，自动收的。我一直放包里，下雨天拿出来用，不下雨就在包里待着。用了四五年了。很好的伞，是那个牌子，叫什么来着，很有名——

她顿了一下，等对方补充，但没等到。

上班族

天堂伞。

上班族

我记得很清楚，因为那天天气预报说有雨。上了车我还看了一眼，就在侧边口袋。然后下车的时候——就没有了。

上班族

不是忘了。我不会忘。我——

5. 地铁里 / 白天 / 手持镜头 — 保洁叫他抬脚

他站在站台，喝瓶装水，喝完了。

保洁阿姨拖着垃圾桶过来了。到他跟前停了一下。

他把瓶子扔到她拖着的垃圾桶里。她继续走了。

6. 站台 / 白天 / 手持镜头 — 采访打工人，丢了一根香蕉

白领

香蕉。

主角（画外）

……香蕉？

白领

对。

停顿。

白领

我哪里都找过了，站台座位，背包里，车厢座位上，全部都没有，警察根本不管。

主角（画外）

你报警了？

白领

当然。

主角

因为丢了一根香蕉。

白领

我买回来之后，正好放了两天一夜，少了皮上会带绿色，再晚几个小时，就会开始长黑点，但那天早上，就那天早上，我起来的时候，它就在那个果盘里，完美，没有一丝绿色，也没有一个黑点。一定是被偷走了。没有生物能抗拒那么完美的香蕉。

主角（画外）

这是哪一天的事？

白领

就前天。

7. 站台座椅旁 / 白天 / 手持镜头 — 采访退休老师，袋子没破但鱼没了

一个退休老师。六十多岁，头发花白，穿着一件看不出颜色的薄外套。他坐在站台的塑料座椅上，腿上放着一个塑料袋。

语气很平，没有任何要把这件事讲成故事的意思。

退休老师

从菜场买的鲈鱼。活的。

他提了提手里的塑料袋，示意了一下。

退休老师

就这种袋子，扎好的，里面有水。上车的时候还能感觉到它在动。

他低头看了一眼袋子。

退休老师

到站下车，我就觉得轻了。一看——袋子还在，水还在。

他停了一下。不是为了制造悬念，是在确认自己要说的话。

退休老师

鱼没了。

安静了两秒。

退休老师

鲈鱼清蒸好吃。

他又看了一眼袋子，好像到现在还在确认这件事。

退休老师

就是——

主角的手机响了。

老妈

你到家了没有？我跟你说，今天你姨打电话来了——

主角

我现在有事妈，等会再给你打回去。

他挂断。

主角

您接着说。

退休老师

就是没有了。

8. 主角的房间 / 夜 / 屏幕录制 — 搜索黑洞

浏览器。搜索框。

他打字：“地铁 东西 消失”

搜索结果：一些失物招领的帖子、地铁投诉、一个关于"地铁能不能带金鱼"的知乎回答。

他往下翻。点开一个。退出来。又点开一个。

然后他换了一个关键词：“物体 凭空消失”

结果变了。第一页出现了一个科普帖子，标题很长，带着"黑洞"两个字。

他点进去。

页面加载出来。文字、配图、一张黑洞的模拟图。

几秒。

黑洞。

他滚动到一段话。

屏幕上显示：“微型黑洞可能附着于高速移动光源/窄缝气流附近。”

他盯着这行字，停了几秒。

黑洞似乎要把意识吸进去——

老妈（画外）

家里没有盐了！

老妈（画外）

柜子里也没了，你去楼下，那个小超市，买一包回来。千万不要买错了，要那种的。

主角

哪种？

老妈

上次你买的那个不对。

主角

哪个？

老妈（画外）

就那个，你不要买错了。

9. 小超市 / 夜 / 手持镜头 — 买盐

货架。各种盐摆在一排。

他拿起一包盐，旁边有人走过，货架间隙狭窄，他侧身让开。

让一个大爷。

让一个拿篮子的女人。

他排在长长的结账队伍后面，开始看手机。搜索：黑洞 出现 概率

他把页面往下滑——

几个词条：

“意大利面条效应”

搜索结果里有一个视频缩略图：一只猴子坐在打字机前面。他点开了。

视频里，猴子在乱按键盘。字母在屏幕上随机跳出来。

旁白

概率猴子定理。给猴子一台打字机和无限的时间，它能打出莎士比亚。

他意识到了什么。

收银员

三块五。

他拿着盐从超市出来，走了几步，低头看一眼袋子。

又看了一眼。

他转身回去。

10. 地铁车厢 / 夜 / 手持镜头 — 盐没了

他坐在地铁上，去站台继续录素材。塑料袋放在旁边的空座上。

他低头看手机。还在查黑洞。

列车到站，开门。上来几个人。关门。

他继续看手机。

又到站了。开门。关门。

他抬头，看了一眼旁边的座位。

塑料袋没了。

他愣了一下。往座位底下看。没有。往前后排看。没有。

他站起来，走到车门旁边，低头看地面。没有。

11. 主角的家 / 夜 / 固定镜头 — 盐呢

他进门。老妈从厨房探出头。

老妈

盐呢？

主角

……丢了。

老妈

什么叫丢了？刚买的怎么丢了？

他没回答，进了房间。

老妈

算了，明天我自己去买。

12. 主角的房间 / 夜 / 屏幕录制 — 反复看采访素材

电脑屏幕。视频播放器。

画面里是退休老师。他坐在站台的塑料座椅上，提着塑料袋。

退休老师

到站下车，我就觉得轻了。一看——袋子还在，水还在。

他拖动进度条，倒回去。

退休老师

袋子还在，水还在。

又拖回去。

退休老师

鱼没了。

他暂停。盯着画面。

13. 站台 / 又一天 / 白天 / 手持镜头 — 保洁阿姨认出了他

他又坐在站台上。手机对着轨道方向在录。

保洁阿姨从远处推着拖把过来。走到他旁边的时候，她的步子慢了一下。

她看他一眼。他把两只脚都抬起来。

她拖过去了。

14. 菜场 / 白天 / 手持镜头 — 问鲈鱼价，买了便宜的黑鱼

喧闹。叫卖声。水产摊位前面，几个水池子里的鱼在翻。

主角

鲈鱼多少一斤？

鱼贩

三十八。

主角

那边的呢？

鱼贩

十五。要哪个？

他指向那个池子。

15. 地铁车厢 / 白天 / 手持镜头 — 第一次带鱼坐地铁，鱼还在

他坐在座位上。镜头低着，对着他的腿和脚——他的膝盖上放着一个塑料袋，袋子里有水，有一条鱼。

列车到站，开门。关门。

一会后，地铁里人满为患，他手扶袋子，跟着人群摇晃。

一会后，一个胖子坐到他旁边，他被挤醒了。

一会后，傍晚，车厢空了，他昏昏欲睡。

门开，他下车，低头，看袋子。

16. 主角的家 / 客厅 / 夜 / 固定镜头 — 吃饭，老妈劝他去面试

饭桌上。一盘红烧鱼——做得不太好看，家常菜的样子。

老妈

对了，上次你姨打电话来，说她同事那边在招人。

她夹了一块鱼，放碗里。

老妈

做什么的我忘了……好像是什么仓库管理的。我都给你约好了，周五早上再去面一下啊，你天天在家待着，也没事干。

他的筷子停了一下。老妈吃了一口。

主角

鱼怎么样？早上买的，新鲜。

老妈

刺多，下次别买了。

17. 菜场 / 又一天 / 白天 / 手持镜头 — 这次直接买鲈鱼，没问便宜的

同一个水产摊。他指另一个池子。

主角

要那个。

18. 地铁车厢 / 白天 / 手持镜头 — 第二次带鱼坐地铁，鱼还是没消失

同样的角度。膝盖上放着塑料袋。鱼在里面。背后窗外，广告灯在显示视频。

到站了。开门。关门。

他拎着袋子，在车厢里走来走去。

他把袋子放在爱心座位上。

他举起袋子，举到头顶，车厢里的人看着他。

他拎着袋子下车了。

19. 站台 / 傍晚 / 手持镜头 — 袋子漏水，保洁大喊提醒

他坐在站台座位，抱着塑料袋。

他低头，缓缓打开，列车出站，发出轰鸣声。

保洁阿姨在远处，停下来，拄着拖把，显然注意到了什么，抬手指他的袋子，脸色惊慌。

行人闻声驻足。

保洁阿姨大声说了什么，声音被进站的车盖过去了，只看见她的面部动作。

他看见了袋子里的东西。

声音减弱，她的声音浮现。

保洁

别往身上放！你那塑料袋儿漏水！！

他站起来。

裤子湿了一大片。手上的袋子稀稀拉拉漏水。

周围的人都看着他。

20. 主角的家 / 厨房和客厅 / 夜 / 手持+固定镜头 — 摘豆角，说明天有面试

裤子挂在阳台上。

鱼在厨房水池子里。

厨房台面上，一盘毛豆，一盘没剥皮的。他在撕掉两边的丝。旁边是老妈，动作比他快。

老妈自己的剥完了，从他盘子里拿过来一把。

老妈

手慢。明天的面试，千万不要忘了，人家介绍的，一定要好好面，一早就过去，不要晚了，早高峰人多，堵车，门口——。

后面的话变得模糊起来，他只盯着手里的豆子，把毛豆尖掰掉，往下扯，丝断开了。

21. 主角的房间 / 深夜 / 屏幕录制+手持 — 深夜反复看视频，决定出门

啪。关灯。房间暗了。

他躺在床上。

他翻了个身。

他打开手机。

他打开浏览器——上次在超市没看完的那个页面还在。他点开了"意大利面条效应"。

一张示意图：一个人形被拉成一根长条。

他盯着那张图。

几秒。

关掉手机。

黑暗。

他又打开了手机。

他盯着屏幕。

关掉手机。

黑暗。

安静。

安静。

然后——被子掀开的声音。

22. 地铁站 / 深夜 / 手持镜头 — 冲进地铁站赶最后一班车

镜头晃动。脚步声很快。他在跑。

地铁站的灯还亮着，但通道里几乎没有人了。他的呼吸声很重。画面乱成一团——天花板，地面，楼梯，墙上的指示牌，模糊的灯光。

他冲过闸机。

站台上空的。他跑到站台边缘。

远处，隧道里传来列车的声音。灯光从隧道里漫出来。

广播

各位乘客，开往松江南站方向的列车，本次列车为今日最后一班车。

车来了。

门开。

他走进去。

23. 车厢内 / 深夜 / 手持镜头 — 空车厢里看见香蕉

只有他一个人。

白色的灯管。座椅一排，全是空的。

他坐下来。

列车开了。

他的呼吸慢慢平下来。

车窗外面黑的，什么都看不见。

他放心下来，回头，然后——他注意到了什么。

下一节车厢的座位上。

一根香蕉。

他的表情凝重起来。

他站起来，慢慢走过去，到座位旁边。

香蕉就在那里。

他伸出手，手快要碰到那根香蕉了。

列车进了隧道。

车窗外面的黑暗突然被打破——广告灯箱一块一块闪过来。光从窗户涌进车厢，一闪一闪，红的，蓝的，白的，颜色很快。

灯光下，香蕉被举起来，翻面，翻回来，翻面，翻回来。

完美的黄色，从头到尾。没有绿色，没有黑点。

完美的香蕉。

男声

一切皆有可能。

他没有停。推开下一扇门。

又一节车厢。灯管有一盏不亮，明暗交替。座位上有一把深蓝色的伞。天堂伞。

他还在走。再下一节，再下一节，灯越来越暗。

黑暗从车厢尽头漫过来，把他吞噬。

24. 地铁站 / 凌晨 / 固定镜头 — 空站台

列车到站，门打开，无人。

站台空无一人。

25. 主角的家 / 早晨 / 固定镜头 — 老妈没发现房间没人

白天的光从窗帘缝隙里照进来。

走廊。老妈的拖鞋声。

老妈（画外）

早饭放桌上了啊，起来记得吃。

房间里没有人。

床上被子掀开着，枕头上有个凹下去的印子。

26. 地铁站 / 早高峰 / 固定镜头 — 保洁捡起一根意大利面

早上八点的地铁站。

人群从各个入口涌进来。闸机滴滴响，一个接一个。脚步声叠在一起，变成一片连续的、低沉的轰响。LED屏幕上滚动着下一班车的到站信息。一切如常。

镜头是固定的。不再是他的手持视角。拿摄像机的人不在了。

保洁阿姨从画面右侧走进来。蓝色工服，拖把。她在擦站台的玻璃护栏。和之前每一次一样。

她擦到一个位置，停了一下，低头。

一根面条。湿的。在地铁站的地砖上。一根意大利面。

她弯腰捡起来，盯着看。

她似乎发现了什么。

面条被扔进垃圾桶。

【完】
